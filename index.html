About

Halo semuanya, kenalin nama saya I Nyoman Anrasansya Dharma Putra, bisa dipanggil Anra. Umur saya 15 tahun, saya lahir di Denpasar, Bali pada tanggal 09 April 2009. Saya tinggal di Renon di Jl. Tukad Citarum M Blok M No 9B. Saya dulu TK di PAUD Kumara Loka, dan melanjutkan SD di SDN 4 Panjer. Saya melanjutkan SMP di SMP Ganesha Denpasar, dan untuk saat ini saya bersekolah di SMKN 1 Denpasar, tepatnya di Denpasar Utara, Ubung. Sekarang, saya kelas 10 dengan mengambil jurusan Rekayasa Perangkat Lunak (RPL).

Alasan saya memasuki jurusan ini karena peluang karir kerja yang luas dan ketertarikan saya dalam teknologi. Kebetulan, saya juga mengikuti ekstra kurikuler robotik dan paskibra. Saya berharap dapat menjadi bagian dari paskibraka Kota Denpasar, dan jika ada kesempatan, saya ingin melanjutkan hingga paskibraka provinsi atau bahkan nasional. Semoga saya bisa mencapai impian ini, astungkara!

Project

Ini adalah game pertama yang saya buat, dan rasanya luar biasa! Awalnya ragu, tapi prosesnya justru seru dan penuh tantangan. Game ini berjudul "Api Banaspati", terinspirasi dari legenda bola api mistis. Sebagai petualang, kamu harus bertahan hidup dengan menghindari bola api yang semakin cepat dan banyak. Gameplay-nya simpel tapi menegangkan! Dari merancang konsep, menggambar karakter, hingga menulis kode, setiap langkahnya memberi pengalaman berharga. Meski masih sederhana, ini jadi awal perjalanan saya di dunia game development. Ke depannya, saya ingin menambahkan level, power-up, dan mode multiplayer!
